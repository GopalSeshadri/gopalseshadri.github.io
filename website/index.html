Speech Recognition System

The mission was to recognize twenty short word utterances recorded by different people.

1. The frequency data extracted from the audio signal were converted to Mel-frequency cepstral coefficient matrix. As all the audio files were exactly one second long, padding of the data matrix was not neccessary.

2. Used three different Neural Network architecture involving CNN, LSTM and GRU. Achieved the highest accuracy score of 0.941 on the test data using GRUs.

Anomaly Detection

This project was developed to create an unsupervised anomaly detection system for time series data.

1. The data exhibited strong trend, seasonality and hence it was stationarised using differencing. Different data file required different levels of differencing, we used statsmodels package to get the level of differencing needed.

2. Used a multitude of unsupervised techniques like One Class SVM, Isolation Forest, K-Means, and an ensemble by Max Voting to detect Anomalies in the Stationary Time Series dataset. Achieved an overall F1-Score of 90%.

Tolkien In Me

The objective of this project was to build a system to generate text that mimics the works of Tolkien.

1. The raw text data was preprocessed, tokenised and gensim word vectors were created.

2. Developed a LSTM based language model from the entire corpus and then another LSTM based sampling model was created to sample from the language model one word at a time.

Toxic comment classification

The goal of the project was to build a system that identifies toxic comments in social media platforms and categorize them into one or more of the classes: toxic, severe toxic, obscene, threat, insult and identity hate.

1. As part of data pre-processing stopwords, punctuations, blank lines, URLs, hyperlinks were removed. Used NLTK's WordNetLemmatizer to lemmatize the words and used Glove 100d word vectors as embeddings.

2. Used Keras to build a Convolutional Neural Network, Recurrent neural network and Naive bayes SVM. The model outputs were stacked to get the class predictions.

Kuzushiji Hiragana Classifier

The goal of the project was to build a system that identifies clasify calligraphically written Hiragan characters correctly.

1. Developed various LeNet based architectures in Keras to Hiragana characters from highly imbalanced Kuzushiji-49 dataset into one of 49 different classes.

2. The Best performing architecture reached an accuracy of 92%. The best model was added into an application developed using Dash and was hosted in Heroku.

English to Japanese Translator

The goal of the project was to build an English to Japanese Language translator.

1. The input, English language sentences, are tokenized into words and target, Japanese language sentences are tokenized into a character. A token in the Japanese language can be a Hiragana or a Katagana or a kanji character.

2. Created an English to Japanese Language translator using Attention architectures. Tried various preprocessing technique, word vectors and architectures to maximize the BLEU score.

Answering Two Supporting Fact Questions Using Memory Network

The goal of this project is to build a Two Hop memory network to answer two supporting fact questions.

1. As part of data pre-processing stopwords, punctuations, blank lines, URLs, hyperlinks were removed. Used NLTK's WordNetLemmatizer to lemmatize the words and used Glove 100d word vectors as embeddings.

2. Used Keras to build Two-Hop Memory Network Architecture that can answer two supporting fact questions from bAbI dataset from Facebook with an accuracy of 89.2%.
